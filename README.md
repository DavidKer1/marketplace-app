## Dependencies
- React Native
- Expo
- Formik
- Lodash
- Firebase

## Getting Starter
```sh
# Clone repository
git clone https://github.com/DavidKer1/marketplace-app.git

# Install dependencies
npm install

# Create and export default firebaseConfig.js on app/firebase/ with the firebase config

# Run app
npm start
```
